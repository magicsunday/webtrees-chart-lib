# @magicsunday/webtrees-chart-lib

Shared JavaScript chart library for [webtrees](https://www.webtrees.net) chart modules.

Provides SVG export, zoom, overlay, text measurement, and localStorage persistence
used by [webtrees-fan-chart](https://github.com/magicsunday/webtrees-fan-chart),
[webtrees-pedigree-chart](https://github.com/magicsunday/webtrees-pedigree-chart),
and [webtrees-descendants-chart](https://github.com/magicsunday/webtrees-descendants-chart).

## Installation

```bash
npm install @magicsunday/webtrees-chart-lib
```

## Usage

```javascript
import { Storage, ChartZoom, ChartExportFactory, measureText } from "@magicsunday/webtrees-chart-lib";
```
